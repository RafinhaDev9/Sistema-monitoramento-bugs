# 🐞 Sistema de Monitoramento de Bugs

## 📌 Sobre o Projeto

O **Sistema de Monitoramento de Bugs** é uma aplicação desenvolvida para registrar, acompanhar e gerenciar falhas encontradas em sistemas de software.  
O objetivo é organizar os bugs de forma estruturada, facilitando o controle das correções e o acompanhamento das atividades da equipe de desenvolvimento.

Este projeto foi desenvolvido para fins acadêmicos, com foco em versionamento utilizando Git e GitHub.

---

## 🎯 Objetivo

Criar um sistema simples e eficiente para controle de bugs, permitindo:

- Registrar falhas no sistema
- Acompanhar correções
- Definir prioridades
- Controlar status
- Organizar tarefas de desenvolvimento

---

## 🧩 Funcionalidades

- ✅ Cadastro de bugs
- 📋 Listagem de bugs
- ✏️ Atualização de bugs
- 🗑️ Exclusão de bugs
- ⚠️ Definição de prioridade
- 👤 Atribuição de responsável
- 🔄 Controle de status (Aberto, Em andamento, Resolvido)
